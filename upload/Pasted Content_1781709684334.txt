── BUG FIX 1: EBM VAT — RRA COMPLIANT ─────────────────────────────────────────

RRA CORRECT VAT CALCULATION:
  Every product has TWO prices:
    cost_price:      What you paid supplier (buying price)
    selling_price:   What customer pays (TTC — includes VAT)

  From selling_price (TTC), calculate:
    selling_price_ht  = selling_price × (100/118)   [excl. VAT]
    vat_collected     = selling_price × (18/118)    [VAT collected]
    gross_profit_ht   = selling_price_ht - cost_price
    profit_margin_pct = (gross_profit_ht / selling_price_ht) × 100

  EBM Receipt must show:
    Désignation | Qté | PU HT | Montant HT | TVA 18% | Montant TTC

  Monthly VAT Declaration (RRA):
    TVA Collectée  = Sum of all vat_collected from sales
    TVA Déductible = Sum of all VAT paid to VAT-registered suppliers
    TVA Nette à Payer = TVA Collectée - TVA Déductible

  Admin Product Form shows:
    [Cost Price HT] ← what you paid (buying price, excl. VAT)
    [Selling Price TTC] ← what customer pays (incl. VAT)
    Auto-displays: Selling HT / VAT Amount / Gross Profit / Margin %

  Admin VAT Report:
    Monthly summary table for RRA filing
    Export as CSV formatted for RRA submission
    Columns: Date | Receipt# | Customer | HT | TVA | TTC | MRC

      const productMap = {};
      products.forEach(p => productMap[p.id] = p);

class Product(models.Model):
    BADGE_CHOICES = [
        ('bestseller','Best Seller'),('new','New'),
        ('hot','Hot'),('popular','Popular'),('','None'),
    ]
    id                  = models.CharField(max_length=20, primary_key=True)
    name_en             = models.CharField(max_length=200)
    name_fr             = models.CharField(max_length=200)
    name_rw             = models.CharField(max_length=200)
    desc_en             = models.TextField()
    desc_fr             = models.TextField()
    desc_rw             = models.TextField()
    category            = models.ForeignKey(Category,
                            on_delete=models.PROTECT,
                            related_name='products')
    emoji               = models.CharField(max_length=10)
    sku                 = models.CharField(max_length=50, unique=True)

    # ── PRICING (RRA COMPLIANT) ───────────────────────────────
    cost_price          = models.DecimalField(max_digits=12,
                            decimal_places=2, default=0)
    selling_price       = models.DecimalField(max_digits=12,
                            decimal_places=2)
    # selling_price is TTC (includes 18% VAT)
    # selling_price_ht = selling_price * 100/118 (computed property)
    # vat_amount = selling_price * 18/118 (computed property)
    wholesale_price     = models.DecimalField(max_digits=12,
                            decimal_places=2)
    # ── PROPERTIES ───────────────────────────────────────────
    @property
    def selling_price_ht(self):
        return round(float(self.selling_price) * 100 / 118, 2)

    @property
    def vat_amount(self):
        return round(float(self.selling_price) * 18 / 118, 2)

    @property
    def gross_profit_ht(self):
        return round(self.selling_price_ht - float(self.cost_price), 2)

    @property
    def profit_margin_pct(self):
        if self.selling_price_ht == 0: return 0
        return round(self.gross_profit_ht / self.selling_price_ht * 100, 1)

    # ── INVENTORY ─────────────────────────────────────────────
    moq                 = models.IntegerField(default=1)
    stock_qty           = models.IntegerField(default=0)
    low_stock_threshold = models.IntegerField(default=5)

    # ── DISPLAY ───────────────────────────────────────────────
    badge               = models.CharField(max_length=20,
                            choices=BADGE_CHOICES, blank=True)
    is_active           = models.BooleanField(default=True)

    # ── STATS ─────────────────────────────────────────────────
    rating_avg          = models.DecimalField(max_digits=3,
                            decimal_places=2, default=0)
    rating_count        = models.IntegerField(default=0)
    sales_count         = models.IntegerField(default=0)
    view_count          = models.IntegerField(default=0)

    # ── WHOLESALE TIER DISCOUNTS ──────────────────────────────
    tier1_discount      = models.DecimalField(max_digits=5,
                            decimal_places=2, default=2)
    tier2_discount      = models.DecimalField(max_digits=5,
                            decimal_places=2, default=3)
    tier3_discount      = models.DecimalField(max_digits=5,
                            decimal_places=2, default=5)

    created_at          = models.DateTimeField(auto_now_add=True)
    updated_at          = models.DateTimeField(auto_now=True)

    class Meta:
        db_table = 'products'
        ordering = ['-created_at']
        indexes = [
            models.Index(fields=['category']),
            models.Index(fields=['is_active']),
            models.Index(fields=['selling_price']),
            models.Index(fields=['stock_qty']),
        ]

# apps/ebm/services.py

class EBMService:
    TIN = "102345678"
    SDC_ID = "SCS-EBM-001"
    SHOP_NAME = "Samuel Cosmetic Shop"
    VAT_RATE = Decimal("0.18")

    @staticmethod
    def calculate_vat_breakdown(selling_price_ttc):
        """
        RRA requires VAT-inclusive pricing.
        Given TTC price, extract HT and VAT.
        """
        ttc = Decimal(str(selling_price_ttc))
        ht = round(ttc * Decimal("100") / Decimal("118"), 2)
        vat = round(ttc - ht, 2)
        return {"ttc": ttc, "ht": ht, "vat": vat}

    @classmethod
    def generate_mrc(cls, receipt_no, date_str):
        """
        MRC = Machine Receipt Code
        Deterministic hash for RRA verification
        """
        import hashlib
        raw = f"{cls.TIN}{cls.SDC_ID}{receipt_no}{date_str}"
        hash_obj = hashlib.sha256(raw.encode())
        mrc = hash_obj.hexdigest()[:16].upper()
        formatted = '-'.join([mrc[i:i+4] for i in range(0,16,4)])
        return formatted

    @classmethod
    def create_receipt(cls, order):
        """
        Creates EBM receipt with full VAT breakdown.
        Each line item shows HT + VAT + TTC separately.
        """
        from .models import EBMReceipt
        from orders.models import Order

        receipt_no = f"RCP{order.created_at.strftime('%Y%m%d')}" \
                     f"{EBMReceipt.objects.count()+1:05d}"
        date_str = order.created_at.strftime('%Y%m%d%H%M%S')
        mrc = cls.generate_mrc(receipt_no, date_str)

        items_detail = []
        total_ht = Decimal("0")
        total_vat = Decimal("0")
        total_ttc = Decimal("0")

        for item in order.items:
            unit_ttc = Decimal(str(item['unit_price_ttc']))
            qty = item['qty']
            breakdown = cls.calculate_vat_breakdown(unit_ttc)

            line_ht  = round(breakdown['ht'] * qty, 2)
            line_vat = round(breakdown['vat'] * qty, 2)
            line_ttc = round(unit_ttc * qty, 2)

            items_detail.append({
                "name": item['name'],
                "qty": qty,
                "unit_ht":  float(breakdown['ht']),
                "unit_vat": float(breakdown['vat']),
                "unit_ttc": float(unit_ttc),
                "line_ht":  float(line_ht),
                "line_vat": float(line_vat),
                "line_ttc": float(line_ttc),
            })

            total_ht  += line_ht
            total_vat += line_vat
            total_ttc += line_ttc

        receipt = EBMReceipt.objects.create(
            receipt_no=receipt_no,
            order=order,
            mrc_code=mrc,
            customer_name=order.customer_name,
            items_detail=items_detail,
            subtotal_ht=total_ht,
            total_vat=total_vat,
            total_ttc=total_ttc,
            payment_method=order.payment_method,
        )

        order.ebm_receipt_no = receipt_no
        order.mrc_code = mrc
        order.save(update_fields=['ebm_receipt_no','mrc_code'])

        return receipt

    @classmethod
    def export_vat_csv(cls, from_date, to_date):
        """
        Monthly VAT report for RRA submission.
        Format compliant with RRA EBM requirements.
        """
        import csv, io
        from .models import EBMReceipt

        receipts = EBMReceipt.objects.filter(
            created_at__date__gte=from_date,
            created_at__date__lte=to_date
        ).order_by('created_at')

        output = io.StringIO()
        writer = csv.writer(output)

        writer.writerow([
            'Receipt No', 'Date', 'Time', 'Customer',
            'Customer TIN', 'MRC Code',
            'Subtotal HT (RWF)', 'TVA 18% (RWF)',
            'Total TTC (RWF)', 'Payment Method',
            'SDC ID', 'TIN'
        ])

        for r in receipts:
            writer.writerow([
                r.receipt_no,
                r.created_at.strftime('%d/%m/%Y'),
                r.created_at.strftime('%H:%M:%S'),
                r.customer_name,
                r.customer_tin or 'N/A',
                r.mrc_code,
                f"{r.subtotal_ht:,.2f}",
                f"{r.total_vat:,.2f}",
                f"{r.total_ttc:,.2f}",
                r.payment_method,
                cls.SDC_ID,
                cls.TIN,
            ])

        totals = receipts.aggregate(
            total_ht=Sum('subtotal_ht'),
            total_vat=Sum('total_vat'),
            total_ttc=Sum('total_ttc'),
        )
        writer.writerow([])
        writer.writerow([
            'TOTALS', '', '', '', '', '',
            f"{totals['total_ht'] or 0:,.2f}",
            f"{totals['total_vat'] or 0:,.2f}",
            f"{totals['total_ttc'] or 0:,.2f}",
            '', '', ''
        ])

        return output.getvalue()

class APIError extends Error {
  constructor(message, status, data) {
    super(message);
    this.status = status;
    this.data = data;
  }
}

const API = {
  BASE: '/api/v1',
  _token: null,
  _refreshToken: null,

  get token() {
    return this._token ||
      sessionStorage.getItem('sc_admin_token');
  },

  set token(val) {
    this._token = val;
    if (val) sessionStorage.setItem('sc_admin_token', val);
    else sessionStorage.removeItem('sc_admin_token');
  },

  async request(method, endpoint, data, isFormData) {
    const url = `${this.BASE}${endpoint}`;
    const headers = {
      'Accept-Language': localStorage.getItem('sc_language') || 'rw',
    };

    if (!isFormData) {
      headers['Content-Type'] = 'application/json';
    }

    if (this.token) {
      headers['Authorization'] = `Bearer ${this.token}`;
    }

    const options = {
      method,
      headers,
      body: data
        ? (isFormData ? data : JSON.stringify(data))
        : undefined,
    };

    try {
      const response = await fetch(url, options);

      if (response.status === 401) {
        await this._handleUnauthorized();
        throw new APIError('Session expired', 401);
      }

      const json = await response.json();

      if (!response.ok) {
        throw new APIError(
          json.error || json.message || 'Request failed',
          response.status,
          json
        );
      }

      // Cache GET responses for offline support
      if (method === 'GET') {
        this._cache(endpoint, json);
      }

      return json;

    } catch (err) {
      if (err instanceof APIError) throw err;

      // Network error — try cache
      if (!navigator.onLine) {
        const cached = this._getCache(endpoint);
        if (cached) return cached;
      }

      throw new APIError(
        'Ikibazo cy\'inzira. Gerageza nanone.',
        0
      );
    }
  },

  _cache(endpoint, data) {
    try {
      localStorage.setItem(
        `sc_cache_${endpoint}`,
        JSON.stringify({ data, at: Date.now() })
      );
    } catch(e) {}
  },

  _getCache(endpoint) {
    try {
      const raw = localStorage.getItem(`sc_cache_${endpoint}`);
      if (!raw) return null;
      const { data, at } = JSON.parse(raw);
      const age = (Date.now() - at) / 1000 / 60;
      if (age > 30) return null; // 30 min TTL
      return data;
    } catch(e) { return null; }
  },

  async _handleUnauthorized() {
    const refresh = sessionStorage.getItem('sc_admin_refresh');
    if (!refresh) { this._logout(); return; }
    try {
      const res = await fetch(`${this.BASE}/auth/refresh/`, {
        method: 'POST',
        headers: {'Content-Type':'application/json'},
        body: JSON.stringify({refresh}),
      });
      if (res.ok) {
        const data = await res.json();
        this.token = data.access;
      } else {
        this._logout();
      }
    } catch(e) { this._logout(); }
  },

  _logout() {
    sessionStorage.clear();
    if (window.location.pathname.includes('admin')) {
      window.location.href = 'admin.html';
    }
  },

SECTION 15: DEPLOYMENT GUIDE

RAILWAY.APP (Recommended — FREE tier):
  1. Create account at railway.app
  2. New Project → Deploy from GitHub
  3. Add PostgreSQL service
  4. Add Redis service
  5. Set all environment variables from .env.example
  6. Railway auto-detects Procfile
  7. Run migrations: railway run python manage.py migrate
  8. Seed data: railway run python manage.py seed_data
  9. Collect static: railway run python manage.py collectstatic

RENDER.COM (Alternative — FREE tier):
  1. New Web Service → Connect GitHub
  2. Build Command: pip install -r requirements.txt
  3. Start Command: gunicorn config.wsgi:application
  4. Add PostgreSQL + Redis services
  5. Set environment variables

LOCAL DEVELOPMENT:
  1. python -m venv venv
  2. source venv/bin/activate (Mac/Linux)
     venv\Scripts\activate (Windows)
  3. pip install -r requirements.txt
  4. cp .env.example .env (fill in values)
  5. python manage.py migrate
  6. python manage.py seed_data
  7. python manage.py runserver
  8. Open frontend/index.html in browser

  celery -A config beat --loglevel=info
